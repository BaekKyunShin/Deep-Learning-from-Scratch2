밑바닥부터 시작하는 딥러닝 2
==========================

## 스터디 목적

- 캐글 NLP대회 참여를 위한 NLP 학습이 목적입니다.
- [밑바닥부터 시작하는 딥러닝2] 책을 활용하여 스터디를 했으며, 캐글 및 데이콘 대회와 병행하며 남는 시간에 학습했기 때문에 여유를 가지고 천천히 진행했습니다.
